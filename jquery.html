Inclusion Utilisation
Jquery est une librairie JS tres rapide et tres precise creee pae John Resig en 2006 avec une belle devise : Write Less Do More.C’est un outil js qui a ete concu pour simplifier des taches diversifiees en ecrivant moins de code.
  • DOM manipulation
    jQuery a rendu facile l’acces aux elements du DOM en donnant la possibilite de les selectionner et de les modifier
  • Event-handling
    jQuery offre un elegant moyen de gerer une large gamme d’evenements comme le fait que l’utilisateur clique sur un boutton sans avoir a toucher au code html.
  • Animations
    jQuery vient avec beaucoup d’effets d’animations déjà predefinies qu’on peut utiliser sur notre site web
  • Lightweight
    jQuery est une librairie tres legere - environ 19KB en taille Cross Browser Support: jQuery a le support cross-browser et marche bien sur a IE 6.0+, FF 2.0+, Safari 3.0+, Chrome et Opera 9.0+
  • AJAX Support
    jQuery aide beaucoup a developper des sites tres responsives en utilisant la technologie AJAX
