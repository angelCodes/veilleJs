Concepts Domaine
Node.js est une plateforme :
▶ permettant d’écrire des applications;
▶ basée sur le langage JavaScript (et l’interpréteur de Chrome);
▶ adaptée à la programmation de serveur Web;
  • Asynchronous and Event Driven
    Toutes les API de la bibliothèque Node.js sont asynchrones, c'est-à-dire non bloquant. Cela signifie essentiellement qu'un serveur basé sur Node.js n'attend jamais de réponse de la part d'un API pour renvoyer des données. Le serveur passe à l'API suivante après l'avoir appelée et le mécanisme de notification d'événements de Node.js aide le serveur à obtenir une réponse concernant le dernier appel de l'API.
  • Very Fast
    Etant construit sur le moteur JavaScript V8 de Google Chrome, la bibliothèque Node.js est très rapide dans l'exécution de code.
  • No Buffering
    Les applications Node.js ne mettent jamais de données en mémoire tampon. Ces applications vont simplement sortir les données en morceaux.

  • Ou utiliser Node Js?

  • Applications liées aux E / S
  • Applications de diffusion de données (Streaming)
  • Applications Single Page
